# Sistema de Control de Acceso a Laboratorio

Sistema en Java con interfaz gráfica usando JOptionPane.  
Permite registrar usuarios y controlar accesos en laboratorio mediante archivos .txt.

## Arquitectura
- Entidades
- AccesoDatos
- LogicaNegocio
- Presentacion

## Ejecución
Abrir en NetBeans y ejecutar Main.java.

## Tecnologías
- Java
- NetBeans
- Archivos planos (.txt)
- Interfaz gráfica con JOptionPane
